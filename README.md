# Lulu

Lulu - A ruby gem for merging map markers by agglomerative clustering.

## Installation

Add this line to your application's Gemfile:

    gem 'lulu'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lulu

## Usage

    require 'lulu'
